# Hospital Appointment System (Django + SQLite)

This is a simple **Hospital Appointment System** built with **Django**, **SQLite**, and plain **HTML/CSS (Bootstrap)**.

It allows patients to book appointments with doctors and lets staff view and manage upcoming appointments.

---

## Prerequisites

- Python 3.10 or higher
- pip
- (Optional) Virtual environment

---

From the project root directory:

```bash
python -m venv .venv

# macOS / Linux
